# Personal Portfolio

This is a **Personal Portfolio Website** designed to showcase skills, experience, and contact information. It features a clean, modern design and a responsive layout, making it perfect for presenting personal and professional information effectively.

---

## Features

- **Elegant Sidebar**:
  - Avatar image and name display.
  - Contact info: Email, phone, birthday, and location.
  - Links to social media profiles (Facebook, Twitter, Instagram).

- **Main Sections**:
  - **About**: Personal and professional summary for introductory purposes.
  - **Resume**: Detailed career and education achievements.
  - **Skills**: List of proficiencies and key competencies.
  - **Portfolio**: Showcases of projects and work samples.
  - **Contact Form**: Simple form to connect with the owner.

- **Responsive Design**:
  - Built with mobile-first principles.
  - Optimized for readability and usability across all devices.

---

## Technologies Used

- **HTML5**: Markup for structuring the web content.
- **CSS3**: Custom styles for theming and layout in `style.css`.
- **JavaScript**: Support for interactivity (optional in the future).
- **Google Fonts**: Professional typography (e.g., `Poppins` font family).
- **Ionicons**: Scalable icons for social media, contact, and navigation.
